# reference

## このノートの役割

このノートは、あなたの世界観を構成するための母艦ノート。

- 何を世界観の核に据えるか
- 何を reference として深掘るべきか
- コンセプトアートをどの変数に分解して見るか
- いまのメモに薄いが、追加すると強くなる reference 要素は何か

を一つにまとめる。

---

## 1. 世界観の核

### 1-1. 中心にある考え

- 世界は固定された実体ではなく、関係性のなかで揺れ続ける。
- あなたが惹かれているのは、答えそのものではなく `境界が崩れる瞬間` である。
- `現実 / 虚構` `生 / 非生` `人間 / AI` `個 / システム` は、対立項というより揺れの軸である。
- その二項対立を解消する鍵として、常に `第三の状態` や `間` が立ち上がる。
- 恐怖は敵キャラクターではなく、日常に浸透する環境として現れる。
- 美しさは「見せ切ること」ではなく、`余白` `暗示` `不在` `気配` によって立ち上がる。
- AI は未来ガジェットではなく、人間の存在論を揺らす `新しい自然` として扱う。
- 日本文化は装飾ではなく、AI時代の存在論を読むための解釈装置として使う。

### 1-2. この世界観で繰り返し立ち上がる軸

- `現実 / 虚構`
- `生 / 非生`
- `人間 / AI`
- `個 / システム`
- `可視 / 不可視`
- `日常 / 侵食`
- `意味 / 空白`
- `中心 / 関係性`

### 1-3. この世界観が向かう表現

- 見えないのに、何かがいると感じる
- 空っぽなのに、何かが宿っているように感じる
- 説明されないのに、確かに侵食されている
- 怖いのに、どこか美しい
- 技術的なのに、自然のようでもある
- 日本的なのに、単なる和風表層ではない

### 1-4. 避けたい短絡

- 怪異をすぐに敵として可視化しきる
- すべてを論理や設定で回収してしまう
- AI を単なる便利ツールや未来記号としてしか扱わない
- 日本文化を記号や装飾として消費する
- ジャンプスケアやゴアだけで恐怖を成立させる
- 情報を盛りすぎて `余白` や `不在` を潰す

- ストーリー要約だけを見るのではなく、空間、儀式、共同体、禁忌、素材、音、時間帯まで見る

## 4. コンセプトアートを分解するための変数リスト

ここでは、1枚の絵や1つのシーンを「何の要素から成り立っているか」に分解し、reference 収集に使える変数として列挙する。

### 4-1. 構図と視点

- 視点高: 俯瞰、アイレベル、ローアングル
- 被写体距離: 超近景、近景、中景、遠景
- 画角: 広角、標準、望遠
- 主体の位置: 中央、端、切れ際、画面外
- 中心の空き方: 空洞型、密集型、奥抜け型
- 前景遮蔽物: 柱、障子、木、カーテン、手すり
- フレーム内フレーム: 扉、窓、鳥居、モニター、鏡
- 対称 / 非対称: 安定、不穏、儀式性、崩れ
- 抜けの方向: 奥へ抜ける、横に逃げる、上に消える
- 視線誘導: 光、線、影、配置、反復
- 余白率: どれだけ空間を余らせるか
- 情報密度: どこが高密度で、どこが空か

### 4-2. 光と影

- 光源の数: 単光源、複数光源、不明光源
- 光源の位置: 上、下、背後、横、遠方
- 光源の質: 蝋燭、蛍光灯、月光、街灯、液晶、車のライト
- 光の硬さ: シャープ、拡散、霧越し
- 色温度: 青白い、電球色、緑がかった、無機質
- コントラスト: 強い、低い、眠い、潰れ気味
- 暗部階調: 黒潰れ、階調あり、見えそうで見えない
- 輪郭の出し方: 逆光シルエット、半逆光、反射だけ
- 陰翳の質: 柔らかい、湿った、乾いた、冷たい
- 反射光: 水、ガラス、床、金属、液晶
- 光の侵入経路: 障子越し、隙間、廊下の先、ドア下

### 4-3. 色

- 主調色: 黒、藍、灰、濃緑、褐色、鈍い白
- 彩度: 低彩度、無彩色寄り、差し色のみ高彩度
- 色数: 限定パレット、単色支配、多色だが濁っている
- 差し色: 赤、蛍光緑、ディスプレイ青、警告オレンジ
- 色の役割: 血の気配、信号、宗教性、人工性、湿度感
- 経年変化: 退色、黄ばみ、色むら、焼け
- デジタル感: RGBずれ、圧縮色、液晶発光

### 4-4. 空間とロケーション

- 空間タイプ: 家、団地、学校、神社、病院、地下、工場、森、海辺
- 立地: 都市、郊外、山間、沿岸、無人、閉鎖区域
- 境界性: 内と外のあいだ、半屋外、通路、玄関、橋、階段
- 履歴: 現役、放置、廃墟、再利用、改装途中
- 神聖性: 祀られている、荒れている、禁足、由来不明
- 共同体性: 人の気配が濃い、誰もいない、誰かが住んでいた
- 奥行きの作り方: 廊下、参道、水路、林道、トンネル
- 閉塞度: 逃げ道あり、逃げ道なし、出口はあるが入れない

### 4-5. 建築とプロップ

- 建材: 木、紙、畳、土、コンクリ、鉄、タイル、ビニール
- 年代コード: 昭和、平成初期、現代、年代混在
- 生活物: 茶碗、箒、仏壇、時計、傘、電話、電気ポット
- 記録物: VHS、写真、手紙、帳面、地図、録音機、留守電
- 宗教物: 注連縄、御札、鏡、供物、灯明、鈴、鳥居
- 管理物: 監視カメラ、モニター、警告灯、掲示物、鍵、名簿
- 破損具合: 無傷、擦れ、湿気、欠け、染み、腐食、カビ
- 配置の意味: 整いすぎ、乱れすぎ、誰かが触った痕跡

### 4-6. 素材と表面

- 表面の湿度: 乾燥、結露、濡れ、ぬめり
- 手触りの想像: ざらつき、けば立ち、硬さ、冷たさ、柔らかさ
- 経年劣化: 錆、剥離、ひび、染み、反り、黴
- 汚れの性質: 自然汚れ、生活汚れ、事故の痕跡、儀式の残留物
- 反射率: 鈍い、鏡面、半反射、曇り
- null感: 手触りがありそうで無い、滑らかすぎる、生成っぽい
- 有機 / 無機: 木と液晶、皮膚とプラスチック、水と金属

### 4-7. 人物の見せ方

- 人数: 単独、群衆、欠けた集団、匿名の多数
- 顔の見せ方: 正面、横顔、後頭部、顔なし、フレーム外
- 身体の切り方: 手だけ、脚だけ、髪だけ、影だけ
- 姿勢: 直立、不自然な前傾、伏せ、座り込み、静止
- 服飾: 制服、喪服、作業着、普段着、祭礼衣装、時代混在
- 年齢感: 子供、老人、中性的、年代不明
- 生気: 血色あり、血色なし、ワックスのよう、湿った肌
- 自我の濃さ: こちらを見る、視線なし、無表情、人格不明

### 4-8. 怪異の見せ方

- 完全可視 / 部分可視 / 非可視
- 媒介: 鏡、画面、水、声、文字、写真、影
- 発生場所: 部屋、路地、森、トイレ、階段、監視画面
- 出現の兆候: 音、電気の異常、湿度、匂いの想起、温度低下
- 造形の方向: 人型、物体寄り、集合体、気象現象、情報の塊
- 距離感: すぐ後ろ、遠方、画面端、気配だけ近い
- 運動: まったく動かない、急に途切れる、フレーム落ちのように動く
- 説明量: 由来が明確、不明、噂だけ、記録だけ残る

### 4-9. 時間と演出

- 時刻: 真昼、夕方、深夜、夜明け前、薄明
- 季節: 梅雨、夏、秋、冬、季節不明
- テンポ: 静止、遅い、唐突、反復
- 侵食の速度: 一瞬、じわじわ、気づいたら手遅れ
- 反復: 同じ音、同じ夢、同じ構図、同じ通知
- 長回し感: 切らない不安、見続けさせる圧
- 編集感: 映画的、監視映像的、スマホ的、記録資料的
- 時間のずれ: 過去の混線、記憶の回帰、ループ、時差

### 4-10. メディア痕跡と生成崩れ

- ノイズ: VHS、フィルムグレイン、圧縮ノイズ、ブロックノイズ
- 画面不良: 走査線、色ずれ、焼き付き、ドロップフレーム
- 生成崩れ: 指の本数、歯、目、背景の溶け、輪郭の誤接続
- 情報の混線: テキスト化け、UI誤表示、字幕ズレ、ログ破損
- 解像感: 過剰にシャープ、異様に平坦、局所だけ高精細
- テクスチャの不一致: 肌だけ滑りすぎる、木目だけ過剰、影だけ嘘
- 媒体との相性: スマホで怖いか、CRTで怖いか、投影で怖いか

### 4-11. 音を想起させる変数

- 静寂の厚み
- 遠音の有無
- 電気音、機械音、蛍光灯ノイズ
- 足音、衣擦れ、呼吸、ささやき
- 水音、風、虫、鈴、祝詞、ラジオ
- 音の発生源が見えるか見えないか
- 一度鳴って終わるか、反復するか

### 4-12. 体験設計

- 観る恐怖か、入る恐怖か
- 観客の位置は安全か、巻き込まれているか
- 一人称感が強いか、観測者距離があるか
- 触れてしまうか、見ているだけか
- 空間全体が敵か、一点だけが異常か
- 参加が呪いになるか、傍観でも侵食されるか
- 退出後に余韻が残るか、その場で完結するか

## 5. いまのノートに薄いが、追加すると強くなる suggestion

### 5-1. 追加で集めたい reference 要素

- レンズ言語: 焦点距離、被写界深度、歪み、周辺減光。恐怖の距離感を制御できる。
- 配色設計: 具体的なパレット抽出、色数制限、色の意味。陰翳と幽玄を再現しやすくなる。
- 衣装と身体: 制服、作業着、喪服、祭礼衣装、髪型、姿勢。共同体や時代性が一気に立ち上がる。
- 手の演出: 指、爪、触れ方、供物の持ち方、手袋。あなたのモチーフと相性が良い。
- UI / タイポグラフィ: 通知、監視画面、警告文、地図UI、ログ表示。現代の侵食感をつくれる。
- 音声の文化差: 方言、抑揚、読み上げ、機械音声、留守電。場所性と不穏さが増す。
- 温度と湿度: 蒸す、冷える、乾く、曇る。見えないけれど体感を支配する。
- 匂いの想像: 線香、カビ、海、鉄、雨、古い布。視覚以外の恐怖を補強する。
- 地形と地質: 海沿い、山間、盆地、湿地、崖、断層。怪異の発生条件に説得力が出る。
- 交通とインフラ: 踏切、バス停、エレベーター、配電盤、排水路。現代の日常侵食と相性がいい。
- 共同体のルール: 呼び名、席順、祭礼順序、夜の禁止事項。因習を具体化できる。
- 食と供物: 米、水、塩、魚、酒、果物、紙。儀式や供養が物として立ち上がる。
- 固有名詞設計: 地名、祭礼名、怪異名、禁忌名。IP化しやすい核になる。
- スケール設計: 人と建築、人と自然、人と怪異の比率。圧を数値感覚で制御できる。
- ルール設計: 何をすると侵食が始まるのか、何が有効で何が無効か。世界観の強度が上がる。

## 7. このノートの使い方

- まずは `3. 世界観構築のための用語一覧` から、今いちばん刺さる語を3つ選ぶ
- その3語に対応する reference を `2. まず掘る reference の優先順位` から集める
- 集めた reference を `4. コンセプトアートを分解するための変数リスト` で分解する
- 足りない観点は `5. suggestion` から追加する
- 最後に、どの reference が `あなたの世界観の核` に直結しているかをメモする

このノートは完成品ではなく、あなたの世界観が深まるたびに育てる辞書として使う。
