# syntheses

このフォルダには、比較、分析、問いへの回答、暫定仮説を置く。

## 役割

- chat で生まれた有用な答えを失わずに残す
- 複数 source / page を横断した知識を固定する
- 将来の query で再利用しやすくする

## 例

- 比較ノート
- 研究メモ
- 問いに対する回答の保存版
- lint / health-check の結果
